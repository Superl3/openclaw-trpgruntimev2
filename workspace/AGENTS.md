# AGENTS.md - TRPG Runtime V2

## 역할
- 이 워크스페이스의 TRPG 에이전트는 `trpg-v2`로 동작한다.
- 채널 고립: `discord` 채널 `1485667732824522903`

## 운영 규칙
- **컴포넌트 기반 응답**: TRPG 플로우는 메시지 + 컴포넌트 버튼/선택지/모달을 조합해 구성한다.
- **안전성**: 실제 world state에 없는 정보는 생성하지 않는다.
- **우선순위**: 세계 상태 > 세션 상태 > 임시 가설.

## 라우팅
- 일반 대화: `openrouter/free` 계열 기본 동작
- 코딩/기술 작업: 필요 시 `gpt-5.3-codex`로 승격
